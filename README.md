Sistem analisis probabilitas juara Liga Inggris menggunakan metode Distribusi Binomial untuk prediksi akurat.

Fitur Utama

- Tambah/Edit/Hapus data klub
- Kalkulasi probabilitas otomatis
- Visualisasi dengan progress bar
- Mode fallback (Session/MySQL)
- Responsive design
- Validasi input real-time

Instalasi

Prasyarat
- PHP 7.4 atau lebih tinggi
- MySQL atau SQLite
- Web server (Apache/Nginx)
